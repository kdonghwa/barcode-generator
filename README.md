# 🏷️ 무료 온라인 바코드 생성기 (Free Online Barcode Generator)

설치 없이 브라우저에서 바로 사용할 수 있는 간편한 바코드 생성기입니다. Code128 및 Code39 포맷을 지원하며, 생성된 바코드를 PNG 이미지로 다운로드할 수 있습니다.

## ✨ 주요 기능

*   **바코드 생성**: Code128 (권장) 및 Code39 포맷 지원
*   **이미지 다운로드**: 생성된 바코드를 고화질 PNG 이미지로 저장
*   **히스토리 관리**: 생성한 바코드 이력이 자동으로 저장되며, 클릭하여 다시 불러오거나 삭제할 수 있습니다.
*   **순서 변경**: 드래그 앤 드롭으로 히스토리 아이템의 순서를 자유롭게 변경할 수 있습니다.
*   **다국어 지원**: 한국어, 영어, 중국어, 일본어, 스페인어 5개 국어를 지원합니다.
*   **완전 무료**: 별도의 설치나 회원가입 없이 브라우저에서 즉시 실행됩니다.

## 🚀 사용 방법

1.  `index.html` 파일을 웹 브라우저(Chrome, Edge, Safari 등)에서 엽니다.
2.  바코드 내용을 입력합니다. (예: ITEM-12345)
3.  바코드 타입(Code128 또는 Code39)을 선택합니다.
4.  **바코드 생성** 버튼을 클릭합니다.
5.  생성된 바코드를 확인하고 **이미지 다운로드** 버튼을 눌러 저장합니다.

## 📦 설치 및 실행

이 프로젝트는 정적 웹페이지로 구성되어 있어 별도의 빌드 과정이나 서버가 필요하지 않습니다.

1.  저장소를 클론하거나 다운로드합니다.
2.  `index.html` 파일을 더블 클릭하여 브라우저에서 실행합니다.
